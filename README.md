# Sales Analytics Dashboard (Excel)

##  Project Overview
This project analyzes sales data to uncover insights related to revenue trends,
customer behavior, product performance, and delivery efficiency.

The dashboard was built using **Microsoft Excel with Power Query and Power Pivot**.

---

##  Tools & Technologies
- Microsoft Excel
- Power Query (Data Cleaning & Transformation)
- Power Pivot (Data Modeling & Measures)
- Pivot Tables & Slicers
- Data Visualization & KPI Analysis

---

##  Key Metrics
- Total Revenue: ₹35+ Lakhs
- Average Customer Value: ₹3,520+
- Average Order-to-Delivery Time: 5.5 days

---

##  Insights Generated
- Seasonal revenue peaks in February and August
- Anniversary and Raksha Bandhan are top-performing occasions
- Colors and Soft Toys contribute the highest revenue
- Strong order volume from tier-2 and tier-3 cities
- Higher revenue observed on Sundays and Tuesdays
